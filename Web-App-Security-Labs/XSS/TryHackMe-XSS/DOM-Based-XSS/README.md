# DOM-Based XSS

## Overview

DOM-based XSS occurs when the vulnerability exists entirely in client-side JavaScript.

The server does not process the malicious input.

## Vulnerability Type

Cross-Site Scripting – DOM-Based

## Key Concept

The vulnerability is caused by unsafe handling of user input in the browser DOM.

## Skills Practiced

- analyzing JavaScript behavior
- identifying unsafe DOM manipulation
- crafting payloads in URLs
- understanding client-side execution
