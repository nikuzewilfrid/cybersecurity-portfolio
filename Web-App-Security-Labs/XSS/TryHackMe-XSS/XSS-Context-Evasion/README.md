# XSS Context & Evasion

## Overview

This section focuses on understanding how XSS payloads behave in different contexts and how attackers bypass security filters.

## Key Concept

XSS payloads must be adapted depending on where the input is reflected:

- HTML context
- Attribute context
- JavaScript context

## Skills Practiced

- analyzing injection context
- crafting advanced payloads
- bypassing filters
- understanding encoding and escaping

## Tools Used

- Burp Suite
- browser developer tools

## Learning Objective

Understand how to adapt payloads to different contexts and bypass input filtering mechanisms.
