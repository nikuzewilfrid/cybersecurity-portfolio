# Cross-Site Scripting (XSS) Labs

Source: TryHackMe

## Overview

This section documents practical Cross-Site Scripting (XSS) attacks performed in a controlled environment.

XSS is a common web vulnerability that allows attackers to execute malicious scripts in a user's browser.

## Types Covered

- Reflected XSS
- Stored XSS
- DOM-Based XSS

## Skills Practiced

- identifying injection points
- crafting malicious payloads
- understanding browser behavior
- exploiting client-side vulnerabilities

## Tools Used

- Burp Suite
- browser developer tools
- manual payload testing

## Learning Objective

Understand how XSS works and how different types of attacks impact users and applications.

## Ethical Note

All testing was conducted in a controlled lab environment.
