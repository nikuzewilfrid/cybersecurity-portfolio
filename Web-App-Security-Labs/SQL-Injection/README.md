# SQL Injection Labs

Source: TryHackMe

## Overview

This section documents practical SQL Injection (SQLi) techniques learned through hands-on labs.

SQL Injection is a critical web vulnerability that allows attackers to manipulate database queries.

## Types Covered

- In-Band SQL Injection
- Blind SQL Injection
  - Authentication Bypass
  - Boolean-Based
  - Time-Based
- Out-of-Band SQL Injection

## Skills Practiced

- analyzing SQL queries
- identifying injection points
- exploiting database vulnerabilities
- understanding database responses

## Tools Used

- Burp Suite
- browser developer tools
- manual payload testing

## Learning Objective

Understand how SQL injection works and how different techniques are used depending on application behavior.

## Ethical Note

All labs were performed in a controlled environment (TryHackMe).
