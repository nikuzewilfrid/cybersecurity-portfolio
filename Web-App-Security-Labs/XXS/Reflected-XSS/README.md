# Reflected XSS

## Overview

Reflected XSS occurs when user input is immediately returned in the response without proper sanitization.

This allows attackers to inject scripts that execute in the victim’s browser.

## Vulnerability Type

Cross-Site Scripting – Reflected

## Key Concept

The malicious script is not stored but reflected instantly by the server.

## Skills Practiced

- identifying reflected inputs
- crafting payloads
- analyzing HTTP responses
