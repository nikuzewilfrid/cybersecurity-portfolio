# Linux Fundamentals

This section showcases essential Linux administration and security concepts used in cybersecurity, especially for SOC analysts, system administrators, and penetration testers.

These projects demonstrate skills in:
- User and permission management  
- File system navigation  
- Access control  
- Bash scripting  
- Basic audit workflows  


## Included Projects

### 1. **User and Permissions**  
Practical tasks involving file ownership, groups, permissions, and privilege handling.

### 2. **Bash Audit Toolkit**  
A custom script designed to collect key system information and security-related data for auditing.


## Tools Used
- Bash  
- Linux CLI  
- Core command-line utilities  


## Purpose  
These projects demonstrate practical Linux skills required for real-world security environments, enabling system inspection, permission analysis, and automation.
