# SOC Analysis Projects

This section contains blue-team–focused projects related to log analysis, threat detection, Sysmon configuration, and SIEM workflows.  
Each project simulates tasks commonly performed by Tier 1 / Tier 2 SOC analysts.


## Included Projects

### 1. **Windows Event Log Analysis**  
Deep investigation of authentication events, process creation logs, and potential security anomalies.

### 2. **Sysmon Incident Detection**  
Custom Sysmon configuration and detection logic designed to identify suspicious behavior.

### 3. **SIEM Workflow & Use Cases**  
A conceptual SOC workflow including alert triage, escalation, and rule creation.

- Sysmon

- ELK/SIEM Concepts

These projects demonstrate my ability to analyze logs, detect anomalies, understand attack patterns, and document findings professionally — essential skills for SOC environments.
