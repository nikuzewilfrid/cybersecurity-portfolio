# SIEM – Wazuh

## Objectif
Installer et configurer un SIEM open-source afin de détecter des activités suspectes.

## Environnement
- Wazuh Manager
- Wazuh Agent (Windows & Linux)
- Dashboard Kibana

## Étapes

### ✔ Installation du Manager
curl -s https://packages.wazuh.com | bash

### ✔ Installation des agents
- Windows MSI
- Linux DEB

### ✔ Détection brute-force SSH
- Génération d’alertes
- Analyse dans Kibana
- Correlation rules
