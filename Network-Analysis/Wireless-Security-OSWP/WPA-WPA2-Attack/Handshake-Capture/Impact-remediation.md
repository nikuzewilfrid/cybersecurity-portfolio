# Security Impact

If exploited in real environments, attackers could:

- gain unauthorized access to the network
- intercept traffic
- launch further attacks inside the network

# Root Cause

Weak WPA/WPA2 passwords and lack of monitoring.

# Remediation

- use strong passwords (12+ characters)
- use WPA3 when possible
- disable WPS
- monitor deauthentication attacks
- implement network segmentation
