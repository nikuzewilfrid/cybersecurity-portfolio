# Wireless Security (OSWP Style)

## Overview

This section demonstrates practical wireless network security testing techniques inspired by OSWP training.

Wireless networks are commonly used and often misconfigured, making them a critical attack surface.

## Topics Covered

- WEP Cracking
- WPA/WPA2 Attacks
- WPA Enterprise Attacks

## Skills Practiced

- packet capture and analysis
- wireless network exploitation
- password cracking techniques
- understanding wireless protocols

## Tools Used

- aircrack-ng suite
- airodump-ng
- aireplay-ng
- hashcat

## Learning Objective

Understand how wireless networks can be attacked and how to secure them.

## Ethical Note

All tests were performed in a controlled lab environment.
